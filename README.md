# 隐私政策
**更新日期：2026年3月24日**
**生效日期：2026年3月24日**

烟台云都海鹰无人机应用技术有限公司（以下简称“我们”）作为“植保联盟农户端”应用程序（以下简称“本应用”）的运营者，深知个人信息对您的重要性，并会尽全力保护您的个人信息安全可靠。本隐私政策将向您说明我们如何收集、使用、存储、共享和保护您的个人信息，以及您如何行使您的权利。

请您在使用本应用前，仔细阅读并充分理解本政策。如您不同意本政策中的任何内容，请立即停止使用本应用。

## 一、我们收集的个人信息
在您使用本应用的过程中，我们会出于以下目的收集您的个人信息：

| 业务场景 | 个人信息类型 | 收集目的 |
|---------|------------|---------|
| 账号注册与实名认证 | 姓名、身份证号、手机号、昵称 | 识别用户身份，完成合规实名认证核验，保障账号安全与交易合规，为您提供地块管理、作业下单等核心功能 |
| 地块数字化管理 | 地块位置信息、经纬度、边界坐标、种植作物品类、种植面积、作物生长周期信息 | 为您提供种植地块的数字化建档与全周期管理，支撑作业需求发布、植保服务精准匹配 |
| 地图与位置服务 | 精确位置信息、经纬度 | 为您显示地块位置、匹配周边认证植保服务资源、查看作业进度与完成轨迹 |
| 设备安全保障 | 设备标识符（IMEI/Android ID/OAID）、设备型号、操作系统版本 | 保障应用安全运行，防止恶意攻击与账号风险，完成设备兼容性适配 |
| 应用分析与优化 | 应用使用记录、点击事件、页面访问时长 | 分析用户使用行为，优化功能体验，提升应用运行稳定性 |
| 问题诊断 | 崩溃日志、错误日志 | 排查应用故障、修复功能问题，持续优化应用运行表现 |

**特别说明**：
- 位置信息仅在您使用地块管理、地图查看、作业服务匹配相关功能时收集，您可以随时通过系统权限管理关闭位置权限，关闭后仅影响相关位置功能，不影响您使用本应用其他核心功能。
- 实名认证信息仅用于法定身份核验与交易合规保障，不会用于无关商业用途，严格遵循最小必要原则。

| 高德地图SDK | 高德软件有限公司 | 位置信息、设备标识符、网络状态 | 提供地图显示、地块定位、位置服务匹配、作业轨迹查看功能 |

**注**：您可以在本应用“设置”中查看完整SDK列表及对应隐私政策链接。如后续新增接入第三方SDK，我们会通过公告、弹窗等方式提前告知您，并征得您的授权同意。

## 三、数据存储与跨境传输
1. **存储地点**：我们收集的您的个人信息，将全部存储在中华人民共和国境内的服务器中。
2. **存储期限**：我们仅在为实现本政策所述目的所必需的最短时间内保留您的个人信息；账号注销后，我们将按法律法规要求完成对应信息的删除或匿名化处理，法律法规另有强制留存要求的除外。
3. **跨境传输**：我们不会将您的个人信息传输至境外。如未来因业务需要发生跨境传输，我们会单独向您告知信息出境的目的、接收方、安全保障措施等内容，单独征得您的授权同意，并严格履行相关法律程序。

## 四、我们如何保护您的信息
我们使用符合业界标准的安全防护措施保护您的个人信息，包括但不限于SSL传输加密、数据存储脱敏、访问权限分级控制、安全审计等技术与管理手段，全方位防止您的个人信息遭到未经授权的访问、公开披露、使用、修改、损坏或丢失。

## 五、您的权利
您可以通过以下方式管理您的个人信息：
1. **查询/更正/补充**：您可以在应用内“个人中心”查看、修改或补充您的账号信息、实名认证信息、绑定手机号、地块信息等内容。
2. **删除**：您可以自主删除已创建的地块信息、历史订单记录等非必要信息。若需删除个人实名信息，请联系我们进行处理。
3. **撤回同意**：您可以通过手机系统设置，关闭位置、存储等系统权限，撤回对相关个人信息收集的同意；撤回同意后，我们将不再收集对应信息，但不影响此前基于您的授权开展的信息处理行为。
4. **注销账号**：如需注销账号，请发送邮件至 `yunduhaiying@163.com` 或通过应用内客服渠道联系我们，我们将在15个工作日内完成核验与注销处理。
5. **获取个人信息副本**：您可以通过上述联系方式联系我们，申请获取我们持有的您的个人信息副本。

## 六、未成年人保护
本应用主要面向从事农业种植生产的成年人。我们不会主动收集未成年人的个人信息。如您为未成年人，请在监护人的陪同与指导下使用本应用；如我们发现未经监护人同意收集了未成年人个人信息，会立即对相关信息进行删除处理。

## 七、隐私政策的更新
我们可能会不时修订本隐私政策。当政策发生重大变更时，我们会通过应用内弹窗、公告等方式通知您，并重新征得您的同意。本政策更新后，若您继续使用本应用，即视为您已充分阅读、理解并同意更新后的隐私政策。
